# Alternative-Huawei-website-design


🔴 Huawei Alternative Design Concept
A high-fidelity, responsive web design project inspired by Huawei’s global brand identity. This project focuses on advanced CSS techniques, including custom shapes, interactive hover states, and immersive video backgrounds to create a modern tech-showcase experience.

🎨 Design Philosophy
Color Palette: Dominated by Red and Black, reflecting the iconic Huawei branding.

Geometric Innovation: Replaced traditional rectangular layouts with custom Hexagonal Divs for a futuristic look.

Immersive Media: Each category features a Video Background to provide a dynamic user experience.

Interactive UI: Extensive use of hover effects and box-shadow to create depth and responsiveness.

🚀 Key Features
Smart Navigation Bar: * Set at 0.8 opacity for a sleek look.

Becomes fully opaque and reveals detailed menus upon hover.

Categorized Content: Five main sections accessible from the top nav:

Huawei Hakkında (About)

Haberler (News)

Ürünler (Products) - The most extensive section.

Sürdürülebilirlik (Sustainability)

Eğitim (Education)

Advanced Product Catalog: * Showcases Phones, Tablets, Smartwatches, Earphones, Modems, and Computers.

Quick-View: Hover over a product to see short specs and shadows.

Deep-Dive: Click any product to navigate to a dedicated detail page.

🛠️ Tech Stack
HTML5: Semantic structure for the library and product pages.

CSS3: * Shapes: Hexagonal clipping/transformations.

Interactions: Hover transitions and opacity management.

Effects: Custom shadowing and transparency.
